前置作業

WHY Python

現在是資訊爆炸的社會 人人都該學習如何寫程式
程式語言五花八門 選擇非常多樣
有C、Python、JS、Php甚至更多
而就入門而言 Python是個適合新手學習的語言

Python是種直譯式語言 也就是寫一行就會執行一行
不用整個寫完再編譯執行
而且語法簡單又短 寫起來很快
也有非常多寫好的Function和Package
省去很多自己手刻的時間
